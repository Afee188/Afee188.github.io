Decor.Co – Home Decor Shopping Website

Decor.Co is a responsive home decor shopping website developed using HTML, CSS, and JavaScript.

Decor.Co – Home Decor Shopping Website Project Description Decor.Co is a responsive home decor shopping website developed using HTML, CSS, and JavaScript. The website showcases a variety of home decor products and provides users with an attractive and easy-to-navigate interface for exploring decorative items. It includes multiple pages such as Home, About, Shop Now, and Contact, allowing visitors to learn about products, browse collections, and get in touch easily.

Tools & Technologies Used

  • HTML5
  • CSS3
  • JavaScript
  • Visual Studio Code

Key Features

  • Responsive and user-friendly design
  • Home page with featured home decor items
  • Shop Now page displaying product collections
  • About page providing website and business information
  • Contact page for customer inquiries
  • Smooth navigation between pages
  • Interactive "Shop Now" button
  • "Go to Top" button for improved user experience
  • Product cards with images and details
  • Clean and modern UI design

Skills Demonstrated
